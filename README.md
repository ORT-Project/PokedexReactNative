## Installation du projet

Cloner le projet.

```bash
  git clone git@github.com:ORT-Project/PokedexReactNative.git
```

Aller dans le projet.

```bash
  cd PokedexReactNative
```

Installer les dependences.

```bash
  npm install
```

Demarrer le server.

```bash
  npm start
```

Demarrer les tests unitaires.

```bash
  npm test
```
